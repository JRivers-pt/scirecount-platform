# ScireCount Platform

## Overview
ScireCount is a premium people-counting dashboard designed for Milesight VS125 sensors.

## Tech Stack
- React (Vite)
- Tailwind CSS v4 (Dark Tech Aesthetic)
- Lucide React (Icons)

## Setup & Run
1. Open terminal in this folder.
2. Install dependencies (if you haven't):
   ```powershell
   npm install
   ```
3. Run the development server:
   ```powershell
   npm run dev
   ```
4. Open browser at `http://localhost:5173`.

## Features
- **Login**: Custom glassmorphism design.
- **Dashboard**: Real-time metrics for Visits, Capacity, and average Stay Time.
- **Data**: Mock integration for Milesight VS125 sensors.
